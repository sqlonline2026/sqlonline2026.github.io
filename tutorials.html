SQL SELECT Statement

The SELECT statement is used to fetch data from a database table.

The SQL SELECT statement allows you to retrieve specific columns or all columns from a table. 

SELECT * FROM employees;

SELECT name, salary
FROM employees
WHERE salary > 50000;

SELECT is the foundation of SQL and is used in almost every query.

SQL WHERE Clause

The WHERE clause is used to filter records based on conditions.

WHERE helps you extract only the records that satisfy a given condition. 

SELECT * FROM students
WHERE marks > 70;

SELECT * FROM employees
WHERE department = 'IT';

Conditions can use operators like =, >, <, AND, OR, IN, BETWEEN.

SQL JOINs

JOINs are used to combine rows from multiple tables.

JOINs allow you to retrieve related data stored across tables. 

SELECT e.name, d.department_name
FROM employees e
JOIN departments d
ON e.department_id = d.id;

Common JOIN types: INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN.
